# rns-image-hosting
image hosting for reticulum network.

The program will run a nomadnet node and a lxmf node. Sending an image to the lxmf node will generate a nomadnet link.

update: An RFS node will also be running now, and you can retrieve files using the same hash.
## How to set it up
```
pip install -r .\requirements.txt
python main.py
```
